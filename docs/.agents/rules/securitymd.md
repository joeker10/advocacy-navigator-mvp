Integrated Advocacy Framework Rules
Verified Document-First: Every data point extracted must be auditable back to the source PDF.

Tech Stack: Initial build as a Web-based PWA. Use Capacitor for the mobile roadmap integration (Android/iOS) later. 

Compliance: All workflows must adhere to HIPAA and FERPA standards described in @docs/framework_spec.md. 

Persistence: Use a hybrid approach for the "Persistent Toggle" (LocalStorage for UI preferences, Database for verified data states). 

Contact Data: Use only the LDAH and SPIN contact information provided in the spec.
